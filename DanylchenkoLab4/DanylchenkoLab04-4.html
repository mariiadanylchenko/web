Основні веб-технології

Глосарій

HTML
Мова гіпертекстової розмітки, що використовується для створення вебсторінок.
CSS
Каскадні таблиці стилів, які визначають вигляд вебсторінок.
JavaScript
Мова програмування для створення інтерактивності на вебсторінках.
HTTP
Протокол передавання гіпертексту, що забезпечує зв’язок між клієнтом і сервером.
